üîë Authentication Instructions

To use protected endpoints, follow these steps:

  1. Use the POST /auth/login endpoint to obtain an access token
  2. Click the "Authorize" button above
  3. Enter your token in the format: Bearer YOUR_ACCESS_TOKEN
  4. Click "Authorize" to apply the token to all requests

Note: Tokens expire after 30 minutes. Use POST /auth/refresh to get a new token.
